## Що таке GitHub?

**GitHub** — це хмарна платформа для зберігання коду та співпраці над проєктами на основі системи контролю версій **Git**.

- **Git** — система контролю версій (відстежує зміни у файлах)
- **GitHub** — веб-сервіс для зберігання Git-репозиторіїв онлайн

---

## Основні поняття

| Термін | Опис |
|--------|------|
| **Repository (репо)** | Папка проєкту з усією історією змін |
| **Commit** | Збережена версія змін з описом |
| **Branch (гілка)** | Окрема лінія розробки |
| **Merge** | Злиття гілок разом |
| **Pull Request** | Запит на злиття гілки |
| **Clone** | Копіювання репо на свій комп'ютер |
| **Fork** | Особиста копія чужого репо |

---

## Основні команди Git

```bash
# Ініціалізація нового репозиторію
git init

# Клонування існуючого репо
git clone https://github.com/user/repo.git

# Перевірка статусу файлів
git status

# Додавання файлів до індексу
git add .

# Створення коміту
git commit -m "Опис змін"

# Відправка змін на GitHub
git push origin main

# Отримання змін з GitHub
git pull

# Перегляд гілок
git branch

# Створення нової гілки
git checkout -b назва-гілки

# Переключення між гілками
git checkout main

# Злиття гілок
git merge назва-гілки
```

## Як почати роботу з GitHub

### Крок 1 — Реєстрація
1. Перейти на [github.com](https://github.com)
2. Натиснути **Sign up**
3. Ввести email, пароль, username
4. Підтвердити email

### Крок 2 — Створення репозиторію
1. Натиснути кнопку **New** (зелена)
2. Вказати назву репозиторію
3. Вибрати **Public** або **Private**
4. Поставити галочку **Add README**
5. Натиснути **Create repository**

### Крок 3 — Перший коміт через веб-інтерфейс
1. Відкрити файл у репозиторії
2. Натиснути іконку олівця (Edit)
3. Внести зміни
4. Натиснути **Commit changes**
5. Написати опис коміту → **Commit changes**

### Крок 4 — Робота через термінал
```bash
git clone https://github.com/MykolaKushmilov/praktyka-opz-2026.git
cd praktyka-opz-2026
# ... вносимо зміни ...
git add .
git commit -m "Мій перший коміт"
git push origin main
```
